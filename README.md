## Instalasi

Langkah-langkah untuk menginstal aplikasi/bank Anda:

1. Clone repositori ini: `git clone https://github.com/HanzYiz/bank.py`
2. Buka direktori: `cd bank.py`
3. Jalankan aplikasi/bank: `python bank.py`
